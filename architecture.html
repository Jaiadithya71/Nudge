System Architecture

📍 Project Nudge

A Git‑synced remote accountability agent & digital twin. Control your laptop, log progress, query databases, and run weekly audits — all from Telegram.

System Overview

Capabilities

What you can do

Every message is parsed by Gemini AI into one of these intents. You don't need to memorize commands — just type naturally.

📝

Log Activity

Log study hours, work sessions, or any activity. Gemini parses the duration and topic, updates your markdown plan table, and pushes to Git.
Example message "studied AWS ML prep for 1.5 hours"
📊

Query Progress

Ask about your weekly hours, streaks, remaining targets, or any plan data. The bot reads your plan and responds with a formatted summary.
Example message "how many hours have I studied this week?"
📸

Desktop Screenshot

Capture a live screenshot of your laptop screen remotely. The local daemon takes the screenshot and sends it back through the bot.
Example message "take a screenshot of my desktop"
⌨️

Run Shell Commands

Execute any shell command on your laptop remotely. The output is captured and sent back to you on Telegram.
Example message "run ls -la on my laptop"
💾

Database Query

Look up student records from the local SQLite database. Returns photos, academic info, and details — all from a Telegram message.
Example message "find student Ravi's photo"
🔍

Weekly Audit

Every Sunday night, an AI-powered audit reviews your week. It calculates Pass/Fail, tracks streaks, and generates actionable insights.
Example message "run my weekly review"
🗓️

Daily Plan

Ask the bot to generate today's plan. Gemini analyses your progress, remaining goals, and deadlines to create an optimized schedule.
Example message "what should I do today?"
💬

Casual Chat

Anything that doesn't match a specific intent is treated as conversation. Gemini responds naturally, like a personal AI assistant.
Example message "I'm feeling unmotivated today"
Step-by-Step Flows

Follow the data

Click a flow to see exactly how your message travels through the system, what happens at each step, and what data is produced.

1
You
Send a message on Telegram
You type a natural message like "studied AWS ML prep for 1.5 hours" and send it to @Agent71_1bot.
2
Cloud Bot
Bot receives the message
The always-on Python server catches the incoming message and forwards it to Gemini for parsing.
3
Gemini AI
AI parses intent & data
Gemini Flash processes your message using structured output mode (Pydantic schema) and extracts the intent type and structured data.
{ "intent_type": "log_activity", "log_update": { "study_hours": 1.5, "notes": "AWS ML prep" } }
4
Plan Mgr
Update the plan table
The Plan Manager service inserts a new row into the accountability plan's markdown table, recalculates weekly totals and week-ending dates.
5
GitHub
Commit & push to repo
The GitHub Service commits the updated plan file and pushes to the main branch.
git commit -m "Log: AWS ML prep [skip ci]" git push origin main → Success
6
Local Daemon
Pull & sync to laptop
The local sync daemon (polling every 60s) detects the new commit, pulls changes, and mirrors the updated plan to your Next_Move/ folder.
7
You
Confirmation received
The bot sends you a confirmation message on Telegram with the logged entry details and your updated weekly total.
1
You
Ask about your progress
You send a message like "how many hours this week?" or "am I on track?"
2
Gemini AI
Intent classified as query_plan
Gemini recognizes this as a progress query and extracts what information you're asking for.
{ "intent_type": "query_plan" }
3
Plan Mgr
Read current plan data
The Plan Manager reads the current week's entries from the accountability plan, calculating hours logged, remaining targets, and streak information.
4
You
Formatted summary delivered
The bot sends a clean, formatted response with your hours, progress percentage, streak count, and any relevant notes.
1
You
Request a screenshot
You send "take a screenshot" or "show me my desktop" from Telegram.
2
Gemini AI
Intent: request_screenshot
Gemini parses the message and identifies the intent as a screenshot request.
{ "intent_type": "request_screenshot" }
3
Cloud Bot
Forward to Local Daemon
The bot sends the screenshot request to the local daemon running on your laptop.
4
Local Daemon
Capture screen
The local daemon captures a screenshot of your desktop using the screenshot service, encodes it, and sends it back through the communication channel.
5
You
Screenshot delivered
The live screenshot image appears right in your Telegram chat.
1
You
Ask about a student record
You send a message like "find student Ravi's photo" or "lookup roll number 21BCS123".
2
Gemini AI
Intent: query_database
Gemini parses the query, extracts the search parameters (name, roll number, etc.).
{ "intent_type": "query_database", "query": { "name": "Ravi" } }
3
Cloud Bot
Route to Local Daemon
The bot forwards the structured database query to the local daemon on your laptop.
4
Local Daemon
Forward to DB Handler
The daemon hands the query to the database handler module for execution.
5
SQLite DB
Query sastra.db
The DB handler runs a SQL query against the local SQLite database (sastra.db), retrieves the matching student record along with their photo.
SELECT * FROM students WHERE name LIKE '%Ravi%' → 1 record found
6
You
Results delivered
The bot sends the student's photo and academic details back to you on Telegram.
1
Trigger
Audit triggered
Either automatically on Sunday night (scheduled) or manually when you send "run my weekly review".
2
Review Svc
Collect week's data
The Review Service reads all logged entries for the current week from the accountability plan — hours per day, topics covered, targets met.
3
Gemini AI
AI generates review
Gemini analyzes the week's data and generates a detailed review with highlights, areas for improvement, and an overall assessment.
4
Review Svc
Calculate Pass / Fail
The service compares actual hours against weekly targets and records the outcome. Streaks are updated — consecutive passes build your streak count.
Target: 15.0 hrs | Actual: 16.5 hrs Result: ✅ PASS | Streak: 4 weeks
5
GitHub
Commit audit results
The audit results and updated plan are committed and pushed to the GitHub repo for permanent record.
6
You
Detailed report on Telegram
You receive a comprehensive audit report — highlights, pass/fail, streak info, and AI-generated improvement suggestions.
1
You
Ask for today's plan
You send "what should I do today?" or "generate my daily plan".
2
Gemini AI
Intent: generate_daily_plan
Gemini identifies this as a daily plan request.
3
Plan Mgr
Read current state
The Plan Manager reads your current progress — hours logged so far this week, remaining targets, upcoming deadlines, and recent momentum.
4
Gemini AI
Generate optimized schedule
Gemini uses your progress data and goals to generate an optimized daily schedule — prioritizing what matters most and balancing workload.
5
You
Receive your plan
The AI-generated daily plan arrives in your Telegram chat, ready to follow.
